Sobre Mim

Olá! Sou um entusiasta da tecnologia e um desenvolvedor full stack apaixonado por transformar ideias em realidade digital. Minha jornada no mundo do desenvolvimento começou com a curiosidade de entender como a web funcionava, e rapidamente evoluiu para uma carreira dedicada a construir soluções completas e eficientes.

Com uma base sólida em HTML, CSS e JavaScript, eu crio interfaces de usuário intuitivas e responsivas. No entanto, minha paixão não para no front-end. Eu mergulho fundo no back-end, trabalhando com bancos de dados (SQL e NoSQL) e desenvolvendo e integrando APIs RESTful para garantir que as aplicações sejam robustas e escaláveis. Minha experiência com WordPress me permite entregar projetos versáteis, desde sites institucionais a plataformas complexas.

Além do desenvolvimento tradicional, uma das minhas grandes especialidades é a automação de processos. Utilizando ferramentas como N8N e Make, eu desenho e implemento fluxos de trabalho inteligentes que otimizam tarefas, conectam diferentes sistemas e liberam tempo para o que realmente importa: a inovação.

Estou sempre em busca de novos desafios e aprendizados, explorando tecnologias que me permitam construir um futuro digital mais conectado e inteligente.

Meus Projetos

A seguir, apresento uma seleção de projetos que demonstram a diversidade de soluções que construo. Cada projeto é único, desenvolvido de forma personalizada para atender às necessidades e dores específicas de cada cliente, desde a automação de processos até sistemas de gestão completos.

Sistema Para Clinica de Fisioterapia

Um sistema de gestão de treinos desenvolvido para a clínica de fisioterapia Reativare. A plataforma permite que os fisioterapeutas criem e gerenciem programas de treino personalizados para cada cliente. Os clientes, por sua vez, podem acessar seus treinos e registrar sua evolução, permitindo um acompanhamento detalhado e eficiente do progresso terapêutico.

HTML5 CSS3 JavaScript React Firebase

Sistema para Lavanderia

Um sistema de gestão completo para a lavanderia Cleanup Shoes. A aplicação foi desenvolvida com um front-end em HTML, CSS e JavaScript, e um back-end robusto utilizando Node.js e Express, com um banco de dados MySQL. O sistema permite o gerenciamento de ordens de serviço, clientes e o status de cada lavagem, otimizando o fluxo de trabalho da lavanderia.

HTML5 CSS3 JavaScript Node.js Express MySQL

Tela de Imóveis

Uma solução completa para um portal de imóveis, utilizando Bitrix24 como back-end e CRM. O projeto envolveu a criação de uma interface rica com filtros dinâmicos para busca de imóveis, construída com HTML, CSS e JavaScript. Desenvolvi customizações diretamente no CRM (SPA 7) e criei uma API intermediária para processar um feed de dados complexo em XML. Um desafio notável foi a implementação de um proxy de imagem para contornar restrições de CORS, assegurando a performance e a correta exibição das mídias.

HTML5 CSS3 JavaScript Bitrix24 CRM API XML Proxy

Distribuição Automática de Leads

Uma poderosa automação para distribuição de leads que utiliza o Firebase como banco de dados. O sistema se integra ao CRM Bitrix24 e oferece um ecossistema completo: os gestores criam a agenda de plantão e analisam um painel de analytics para ver a distribuição dos leads, enquanto os corretores acessam o sistema para visualizar suas próprias agendas. A solução possui registro de logs para auditoria, lógica de distribuição round-robin e uma API para receber leads de múltiplas fontes, garantindo uma atribuição rápida, justa e rastreável.

Node.js JavaScript Firebase Bitrix24 Automação API Logs Dashboard
